HTML UL

Style your ul inside a nav div, then proceed. In unordered list each element in the list is defined using li tag. Syntax: ul content ul Attributes of unordered list: List-style-type: This attribute is used to specifies the bullet style that will be used as the list item marker.

HTML IMAGE

The <img> element is super important. Use src="..." for the source, use the alt="..." for alternative text. You can style inline by using the .div method I outline in css.

HTML ID

The id="..." element. the most specific way to target an element, use #id in css. If youre having trouble adding a style, especially in larger files it may be because an id tag is overridding it. Stay vigilant!

STYLE SHEET

The default html styling will have attributes for your padding and margin, making styling much more difficult. You can reset your stylesheet using * { margin 0; padding 0; }

VARIABLE WIDTH

When using fixed widths, you run the risk of a mobile site that is fully illegible. By using 'vw' sizes you can make each element variable to the width of the page, including margins, text, shapes and more!

NAVIGATION BAR

To style a nav, use the following. First .nav is for styling the bar itself, make this fixed. Next .nav ul is for removing the bullet style using list-style-type. Next .nav ul li is for styling the position of the buttons, use float: left;. Finally .nav ul li a is for styling the individual buttons.

INLINE ELEMENTS

For inline elements, use .row and child .container and style as follows. First .row { display: flex; } and then .container { position: inline-block;

NOWRAP

5. To prevent an element from skipping to extra lines, use the white space command. In css use white-space: nowrap; and to will stay on the defined line.

ADDING FONTS

6. You can add fonts using a stylesheet generated and hosted by Adobe Fonts or Google Fonts. You can also add a font using the @font-face command. Get creative!!

TEXT DECORATION

You want to underline your text? Or maybe you want to strike-through? All of these styles are nestled under the text-decoration attribute in css. Try it out!

TEXT ALIGN

Want to align your text? Use the align-text attribute and specify 'justify' or other and check out the results!
